# FoodHub — Доставка еды

E-commerce: каталог ресторанов, меню с категориями, корзина на Zustand, оформление заказа. Работает без бэкенда.

## Запуск

```bash
cd web-dev-landing/foodhub
npm install
npm run dev
```

Откройте http://localhost:3004

## Технологии

Next.js 16, TypeScript, Tailwind CSS v4, Zustand, Lucide React.

## Переменные окружения

| Переменная | Обязательна | Описание |
|------------|-------------|----------|
| `NEXT_PUBLIC_BASE_URL` | Да | URL для SEO мета-тегов |
