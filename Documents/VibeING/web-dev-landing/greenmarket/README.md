# GreenMarket — Фермерский маркетплейс

Каталог фермерских продуктов с фильтрами, корзиной-drawer и подпиской на регулярные поставки. Работает без бэкенда.

## Запуск

```bash
cd web-dev-landing/greenmarket
npm install
npm run dev
```

Откройте http://localhost:3003

## Технологии

Next.js 16, TypeScript, Tailwind CSS v4, React Context, Zod, Lucide React.

## Переменные окружения

| Переменная | Обязательна | Описание |
|------------|-------------|----------|
| `NEXT_PUBLIC_BASE_URL` | Да | URL для SEO мета-тегов |
