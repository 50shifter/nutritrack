# LuxStay — Бутик-отели

Лендинг для сети бутик-отелей с многошаговой формой бронирования, анимациями и SEO.

## Запуск

```bash
cd web-dev-landing/luxstay
npm install
npm run dev
```

Откройте http://localhost:3005

## Технологии

Next.js 16, TypeScript, Tailwind CSS v4, Framer Motion, GSAP, Lucide React.

## Переменные окружения

| Переменная | Обязательна | Описание |
|------------|-------------|----------|
| `NEXT_PUBLIC_BASE_URL` | Да | URL для SEO мета-тегов |
